# LogiStorm

LogiStorm는 물류센터 운영을 설명/설계/구현하기 위한
Operational Digital Twin 플랫폼이다.

본 프로젝트는 CAD가 아니라,
물류 운영 흐름을 재현하고 병목을 가시화하며
KPI를 계산하는 것을 목표로 한다.

---

## 목표

Level 1:
- 단위 체계 고정
- world schema 정의
- FlowGraph 모델 정의
- 이벤트 및 상태 모델 정의

Level 2:
- Load 이동 재현
- Queue / Gap / Blocked 처리
- 병목 가시화
- 리플레이
- KPI 계산

제어 기능은 Level 2 범위에 포함되지 않는다.
